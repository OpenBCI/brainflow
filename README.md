# Brainflow

Brainflow is a library intended to obtain, parse and analyze EEG\EMG\ECG data from OpenBCI boards.

## [BrainFlow Docs, Dev and User guides and other information](https://brainflow.readthedocs.io)

## Build status
Compiled with:
* MSVC on Windows
* GCC on Linux
* Clang on MacOS
